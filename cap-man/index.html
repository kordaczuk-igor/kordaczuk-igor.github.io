Cap-Man Repository

Overview

Cap-man is a simple academic project developed as part of university coursework, focused on learning proper multithreading in Java.

The game is a basic and intentionally simplified recreation of Pac-Man, designed primarily to demonstrate thread handling.

Key Features

  • multiple map sizes
  • high score saving to file
  • classic Pac-Man style gameplay
  • power-ups:
    • ghost freeze
    • teleportation
    • player speed boost
    • ghost slowdown
